Описание системы
Цели создания системы:
1. Успешное выполнение лабораторных работ
2. Создание удобной системы для контроля склада с помощью роботов
3. Мониторинг технического состояния роботов

Функциональные возможности:
1. Контроль за выполнением задач транспортировки товаров с помощью роботов
2. Полный контроль технического состояния, статуса и позиции робота.
